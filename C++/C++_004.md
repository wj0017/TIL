### 생성자

* 기본 생성자
* 다중 정의된 생성자(매개변수)
    * 변환생성자(매개 변수가 1개)
* 복사 생성자
    *  r-value 참조
* 이동 생성자

핵심적으로 문제 되는 것 -> 임시객체 (코드에서 드러나지 않음)

#### 생성자 위임

```c++

    class Time{
    public:
        Time() : h(0), m(0), s(0) {}
        Time(int s_) : Time(){
                s = s_;
        }
        Time(int m_, int s_) : Time(s_){
                m = m_;
        }
        Time(int h_, int m_, int s_) : Time(m_, s_){
                m = m_;
        }
    }

```

### [복사 생성자] 필요한가?

Deep copy, shallow copy의 차이를 알아야한다

nData를 가르키는 포인터가 여러개가 될때 shallow copy이고 nData 자체가 여러개가 될때에는 Deep copy 이다
